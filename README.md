# Exam

Алгоритм решения задачи:
1. Разделить исходные данные на размеченные (0, 1, 2), тестовые (?) и неразмеченные (-).
2. Разделить размеченные данные на обучающие (70%) и тестовые (30%).
3. Обучить модель логистической регрессии на данных широты и долготы с поиском параметров по сетке, затем на всех данных (широте, долготе и времени). Аналогично поступить со случайным лесом, выбрать модель с большей долей правильных ответов.
4. Воспользоваться случайным лесом, чтобы предсказать вероятности принадлежности к маршрутам неразмеченных данных. К данным с наиболее высокими вероятностями (>0.8) добавить предсказанные маршруты и эти данные добавить к обучающей выборке.
5. Обучить модель случайного леса с лучшими параметрами на новой обучающей выборке.
